This project is a fully functional web calculator built as part of The Odin Project Foundations. It features a custom-built logic engine that handles basic arithmetic, operator chaining for continuous calculations, and rigorous edge-case management, such as preventing multiple decimal points and handling division by zero. The application is built with a focus on clean, modular JavaScript, utilizing event delegation to manage inputs and a decoupled structure that allows for both mouse clicks and full keyboard support. For the layout, I used semantic HTML and a Flexbox-based design with calculated proportions to create a professional, stable user interface.
